# Rounds

## GET rounds for an event (league+season)

- `/event/:key/rounds`

Examples:

- `/event/world.2014/rounds`  -- World Cup Rounds (Brazil 2014)
- `/event/de.2013_14/rounds`  -- Deutsche Bundesliga Rounds (Season 2013/14)
